# diome-api

API REST de tarefas — Node.js + TypeScript + Express.

## Rotas

| Método | Rota         | Descrição              |
|--------|--------------|------------------------|
| GET    | /tasks       | Lista todas as tarefas |
| POST   | /tasks       | Cria uma tarefa        |
| PUT    | /tasks/:id   | Marca como concluída   |
| DELETE | /tasks/:id   | Remove uma tarefa      |

## Como rodar

```bash
npm install
npm run dev
```
